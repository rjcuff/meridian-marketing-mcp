# Weekly Channel Report

A full picture of this week vs. last week across all connected tools.
Paste this into Claude Desktop. It will pull live data and write the report.

---

## Traffic — Google Analytics 4

- Total sessions this week vs. last week (number + % change)
- Sessions by channel: organic search, direct, referral, social, email
- Top 5 landing pages by sessions this week
- Goal completions or conversions this week (total + by type if available)
- Anything that spiked or dropped by more than 20% — flag it

## Search — Google Search Console

- Total clicks this week vs. last week
- Average position this week vs. last week
- Top 5 queries by clicks
- Any pages that lost more than 3 ranking positions week-over-week — list them

## CRM — HubSpot (if connected)

- New contacts created this week
- Deals created vs. deals closed this week
- Any email campaigns sent — list them with open rate and click rate
- Pipeline value change vs. last week

## Comms — Slack (if connected)

Post this completed report to the #marketing channel when done.
Use the format: "📊 Weekly Marketing Report — [date range]"

## Takeaways

After pulling all the data, give me:

1. **Biggest win this week** — one sentence, include the actual number
2. **Biggest problem to investigate** — one sentence, what specifically to look at
3. **One thing to do differently next week** — concrete action, not advice

Keep the full report under one page. Use tables for numbers, bullets for everything else.
